Help & Documentation

How to Use

  1. Navigate to a supported website
  2. Click the extension icon in your browser toolbar
  3. Click "Extract Board Games"
  4. Game names are copied to your clipboard

CSS Selector Guide

CSS selectors are patterns used to find elements on a webpage:

  • h3 - All h3 elements
  • .product-name - Elements with class "product-name"
  • #game-title - Element with id "game-title"
  • article h3 - h3 elements inside article elements
  • [data-id] - Elements with data-id attribute

Filter Patterns

Use regex patterns to filter results:

  • ^Sponsored - Exclude items starting with "Sponsored"
  • Advertisement - Exclude items containing "Advertisement"
  • .*Game$ - Include only items ending with "Game"
