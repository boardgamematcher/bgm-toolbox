Help & Documentation

What it does

  • Shop deals: on supported board game shops, click the toolbar icon to extract game names and prices, then match them against BoardGameMatcher.com.
  • Play history: on Board Game Arena and Yucata, import your past plays into your BGM profile in one click.
  • Wishlist: quickly add games to your BGM wishlist from the popup.

Custom patterns

The Custom Patterns tab lets you teach the extension a new shop. See the site-profiles README for the schema and examples.
